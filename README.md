# Repo-to-Reel Studio

Transform GitHub repositories into video content.

## Quick Start

```bash
# Install dependencies (one time)
npm install
npm run ingest:install

# Start everything with ONE command
npm start
```

That's it! Open http://localhost:8080

## What it does

- Starts backend server (Python/FastAPI on port 8787)
- Starts frontend (React/Vite on port 8080)
- Both run in one terminal window with colored output

## Test it

Go to Dashboard → Paste `facebook/react` → Click "Generate Video"
